Assalamualaikum awak,

Awak masih marahkan saya lagi ke? Saya minta maaf. Saya tahu mungkin saya tak mampu pujuk awak lagi selepas semua yang saya pernah cakap pada awak. Awak betul tentang sikap saya. Selama ini bila awak suruh saya baca balik chat-chat kita, saya tak baca dan tak percaya cakap awak, sebab saya rasa saya yang betul.

Tapi sekarang saya sedar, awak betul bila awak kata saya tak berubah, dan awak juga betul bila awak kata semua orang boleh berubah, cuma nak atau tak nak saja. Awak percaya yang saya boleh berubah, awak sabar dan tunggu. Tapi saya abaikan semua tu. Saya banyak terpengaruh dengan kawan-kawan saya, yang hidup dalam dunia ni. Lelaki perlu kawal segalanya. Tapi sejak saya menyendiri ni, makin lama, makin saya sedar, bukan macam tu cara sebenarnya. Lelaki sepatutnya bertolak ansur, memimpin dan bukan mengawal.

Bila saya cerita pada kawan-kawan yang saya dah putus dengan awak, ramai yang tak ambil kisah pun. Masa tu saya sedar, saya yang berada dalam circles yang salah. Bukan awak. Saya yang banyak ikut perangai orang lain, sampai saya lupa siapa yang benar-benar sayang saya dan mahu saya jadi lebih baik.

Masa saya cuba menyendiri ni, barulah saya betul-betul fikir. Hidup ni tak boleh fikir untuk sekarang saja. Kita kena fikir untuk masa depan juga. Saya banyak fikir pasal diri sendiri sehinggakan, saya lupa untuk ambil berat pada awak.

Awak masih ingat tak kali pertama kita jumpa kat IOI City Mall? Masa tu awak malu-malu lagi dengan saya. Comel sangat. Dari situ saya rasa, awaklah keajaiban paling indah dalam hidup saya. Sejak hari tu, saya selalu stalk awak. Kalau boleh, setiap hari saya nak jumpa awak. Namun apakan daya, lepas tu, Covid datang balik. Walaupun begitu, saya tetap berusaha untuk buat awak happy dengan saya. Yalahkan cinta pertama saya. Saya tenang bila bersama awak. Saya rasa, saya lah orang paling bahagia dalam dunia ni.

Masa tu, niat saya betul-betul ikhlas nak kahwin dengan awak. Saya nak halalkan hubungan ni. Saya mula simpan duit, impian saya pada masa itu, saya nak halalkan awak suatu hari nanti.

Tapi, selepas saya tengok kawan-kawan saya berpegangan tangan dan buat perkara yang tak sepatutnya, saya mula ikut. Saya bawa awak ke jalan yang tak diredhai Allah. Saya ajak awak buat perkara yang salah. Saya dah jauh dari pegangan asal saya. Saya terlalu percaya cakap orang, kononnya kalau nak perempuan tak tinggal, kena buat semua tu. Saya salah.

Saya ikutkan hawa nafsu. Saya abaikan nasihat awak. Saya bangga diri. Apa-apa awak cakap, saya tak peduli. Tapi bila saya ambil masa sorang-sorang dan muhasabah, barulah saya sedar, semuanya salah.

Saya banyak tengok ceramah Ustaz Adnin sekarang. Dia ada cakap yang manisnya cinta tu sebenarnya datang selepas kahwin. Dan saya sedar, hubungan kita ni tak akan ke mana kalau terus macam ni. Saya dah jauh tersasar dari jalan Allah. Saya terpilih jalan yang syaitan suka.

Saya tahu saya bukan lelaki yang baik ataupun sempurna. Tapi kali ini betul, saya akan berusaha untuk menjadi lebih baik. Saya cerita semua benda ni pada mama. Mama nasihatkan saya untuk berubah. Lelaki sepatutnya membimbing wanita ke jalan syurga. Tapi mama juga cakap, lebih baik kalau wanita tu pun sama-sama saling menasihati dan membimbing ke arah kebaikan. Masa tu saya terfikir, awaklah satu-satunya orang yang paling banyak tegur dan bimbing saya ke arah tu.

Kalau saya diberi peluang terakhir, saya nak bimbing awak ke jalan yang benar. Sejujurnya, saya betul-betul rindukan awak. Tapi keputusan semua di tangan awak, sama ada awak masih boleh terima saya lagi ataupun tidak, sebab awaklah orang yang paling berhak untuk pilih siapa yang awak nak bersama.

Tugas pertama saya adalah saya nak minta maaf depan-depan dengan family awak, sebab dah libatkan mereka dalam semua ni.

Saya nak call awak, tapi saya tahu awak mungkin takkan angkat. Terima kasih kerana luangkan masa untuk baca semua ni.
